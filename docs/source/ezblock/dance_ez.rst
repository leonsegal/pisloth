**提示**

PiSloth除了可以播放音效和说话，还可以播放设定好的背景音乐，背景音乐的音量可以调节（0%-100%）。

.. image:: media/bg_volume.png

重复块可以帮助您多次执行相同的代码以减少代码大小。

.. image:: media/dance4.png



**示例**

整个舞蹈分为2个部分，PiSloth会随着音乐完成这2个部分。如果你不暂停代码，它会重复跳舞一直跳下去。
